מדיניות פרטיות ותנאי שימוש

ברוכים הבאים לאפליקציית Collecta. מסמך זה מפרט את אופן איסוף, שימוש והגנה על המידע שלך, וכן את תנאי השימוש באפליקציה.

א. מידע שנאסף

האפליקציה אוספת מידע לצורך מתן שירותי איסוף תרומות, ניהול נהגים ונתיבי איסוף.

  • אימות משתמשים באמצעות Google Sign-In ו-Firebase Authentication.
  • פרטי משתמש בסיסיים: שם משתמש, דואר אלקטרוני, מזהה Firebase ותמונת פרופיל אם זמין.
  • פרטי עסק ותורם: שם העסק, כתובת העסק, פרטי התקשרות.
  • מידע על תרומות: סוגי מוצרים, כמות, זמנים לאיסוף, סטטוס איסוף וכתובות יעד.
  • מיקום גיאוגרפי: בעת שימוש בתכונות מסלול pickup, האפליקציה עשויה לבקש גישה למיקום כדי לחשב מסלולים נכון.

מידע שלא נאסף

האפליקציה אינה אוספת מידע מעבר למה שצויין מעלה ללא קבלת הסכמה מפורשת. אין איסוף אנשי קשר מהמכשיר, SMS או מעקב חיצוני ללא ידיעת המשתמש.

ב. מטרות השימוש במידע

  • הרשמה ואימות משתמשים.
  • ניהול פרופילי תורמים ונהגים.
  • חישוב מסלולי איסוף ותיאום בין תורמים לנהגים.
  • אחסון נתוני תרומות וסטטוס איסוף ב-Firebase Firestore.
  • הצגת מיקומים ושימוש ב-Google Places API לצורך חיפוש וכתובות.

ג. שירותי צד שלישי

האפליקציה משתמשת בשירותים הבאים:

  • Firebase Authentication
  • Google Sign-In
  • Cloud Firestore
  • Firebase Storage
  • Google Places API
  • Google Maps / Geolocation API

ד. אבטחת מידע

הנתונים נשמרים בתשתית Firebase המאובטחת. גישה לנתונים מוגבלת באמצעות מערכת אימות Firebase הרלוונטית, ולא נשתף את המידע עם צדדים שלישיים למטרות פרסום.

ה. זכויות משתמש

  • זכות לתיקון מידע אישי.
  • זכות למחיקת מידע אישי.
  • זכות לקבל מידע על הנתונים הנשמרים לגביך.

לצורך מימוש זכויות אלה, יש ליצור קשר עם צוות הפרויקט.

ו. תנאי שימוש

  1. האפליקציה מוצעת כפי שהיא, ללא אחריות על שלמות או התאמה למטרה מסוימת.
  2. אין להעתיק, לשכפל או להפיץ את התוכן או הקוד של Collecta ללא אישור.
  3. השימוש באפליקציה חייב להיות בהתאם לתנאי השירות של Firebase, Google וכל ספק צד שלישי אחר.
  4. אסור להשתמש באפליקציה לפגיעה בפרטיות של משתמשים אחרים או לפעילויות אסורות.
  5. צוות Collecta רשאי לעדכן את מדיניות הפרטיות ותנאי השימוש בכל עת. שינויים ייכנסו לתוקף מרגע פרסומם.

ז. הסרת מידע

אם ברצונך למחוק את חשבונך או את הנתונים האישיים שלך, יש ליצור קשר עם צוות הפרויקט.
